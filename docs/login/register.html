About OldMa's Game Statistics

This website is designed to view your personal board game statistics on the web. This service is self-hosted, meaning that to use it you need to host it yourself. This can be done locally or on a server.

For more information on how to set up your own instance, please refer to this guide on how to host your own.

Is this already your hosted version but you didn't register your account yet? The step of creating an account is mentioned in the setup guide. This is also the place where you should look when you forgot your password.
