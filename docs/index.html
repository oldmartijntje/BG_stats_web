Board Game Statistics Tracker

Track your board game collection, plays, and scores with this easy-to-use web tool. View comprehensive statistics and insightful graphs for your games, plays, and all players.

This web app was born out of necessity. When I stopped using a smartphone, I encountered a problem: how to continue tracking my board game statistics? The excellent Board Game Stats mobile app allows you to export plays, which you can import here, and vice versa.

Note: This project is not affiliated with the Board Game Stats team, nor is it monetized in any way. If you're a mobile user, I highly recommend trying their excellent app first. This alternative is specifically designed for those who prefer a desktop or web-based solution.

Key Features

  • Import and export compatibility with Board Game Stats app
  • Detailed statistics and visualizations
  • Player performance tracking
  • Game collection management
  • Self-hosted solution for privacy

Technical Details

This application is built with Node.js/TypeScript and MongoDB, which can run either locally or hosted on a server. It includes a login system to keep your data secure when hosted. Best of all, it's completely open source, so you can customize it or add your own features.
